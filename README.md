# 🎬 Movie Dashboard Project

## 📌 Project Overview

This project is an interactive **Movie Dashboard** created using Microsoft Excel.
It provides insights into movie data such as top movies, ratings, and performance using charts and visualizations.

---

## 📊 Features

* Top 5 Movies Visualization
* Movie Ratings Analysis
* Interactive Dashboard (Charts & Graphs)
* Clean and User-Friendly Design

---

## 🛠️ Tools Used

* Microsoft Excel
* Charts (Bar Chart, Pie Chart, etc.)
* Pivot Tables

---

##  Files Included

* `movie_dashboard.xlsx` → Main dashboard file
* `dashboard_preview.png` → Screenshot of dashboard

---

##  How to Use

1. Download the Excel file
2. Open it in Microsoft Excel
3. Explore the dashboard and charts

## 🎯 Purpose

This project is created for learning and demonstrating **data visualization skills using Excel**.
